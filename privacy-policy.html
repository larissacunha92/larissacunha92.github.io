Política de Privacidade — Versículo Diário

Última atualização: 23 de março de 2026

O aplicativo Versículo Diário respeita a sua privacidade e está comprometido com a transparência no tratamento de dados. Esta Política de Privacidade descreve quais informações podem ser tratadas, como são utilizadas e quais são os seus direitos como usuário.

1. Dados coletados

O aplicativo não exige cadastro e não coleta diretamente dados pessoais como nome, e-mail ou telefone.

No entanto, serviços de terceiros utilizados no aplicativo podem processar automaticamente informações como:

  • identificadores do dispositivo
  • endereço IP
  • tipo de dispositivo e sistema operacional
  • identificadores de publicidade (Advertising ID)
  • dados de uso do aplicativo relacionados à navegação e interação com anúncios.

Esses dados são utilizados de forma agregada e não permitem identificação direta do usuário pela desenvolvedora.

2. Armazenamento local no dispositivo

O aplicativo pode armazenar informações localmente no dispositivo, incluindo:

  • versículos favoritos
  • preferências de tema, fonte e tamanho do texto
  • configurações de notificações e lembretes

Esses dados permanecem exclusivamente no dispositivo do usuário e não são compartilhados com terceiros.

3. Publicidade

O aplicativo utiliza o Google AdMob para exibição de anúncios.

O AdMob pode coletar e processar dados como:

  • identificadores de publicidade
  • informações do dispositivo
  • dados de interação com anúncios

O aplicativo também pode exibir anúncios premiados (rewarded ads), que permitem ao usuário desbloquear funcionalidades, como temas visuais, de forma opcional.

Esses anúncios são exibidos apenas mediante ação voluntária do usuário.

Para mais informações sobre como o Google utiliza os dados, acesse: https://policies.google.com/privacy

4. Consentimento e gerenciamento de preferências

Quando exigido pela legislação aplicável, o aplicativo apresenta uma mensagem de consentimento para o uso de dados relacionados à publicidade. O usuário pode:

  • aceitar ou recusar o uso de dados para anúncios personalizados
  • gerenciar suas preferências de privacidade

O consentimento pode ser revisado ou alterado a qualquer momento dentro do aplicativo, acessando: Configurações > Configurações de anúncios

5. Compartilhamento de dados

A desenvolvedora não vende dados pessoais dos usuários.

No entanto, dados técnicos podem ser processados por parceiros de publicidade, como o Google AdMob, exclusivamente para:

  • exibição de anúncios
  • medição de desempenho
  • prevenção de fraudes

6. Segurança

São adotadas medidas razoáveis para proteger as informações tratadas pelo aplicativo contra acesso não autorizado, alteração ou divulgação.

7. Privacidade de crianças

O aplicativo não é direcionado especificamente a crianças menores de 13 anos e não coleta intencionalmente dados pessoais desse público.

8. Base legal (LGPD)

O tratamento de dados é realizado com base na legislação aplicável, incluindo a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

As bases legais aplicáveis incluem:

  • consentimento do usuário, quando exigido para publicidade personalizada
  • legítimo interesse, para funcionamento básico do aplicativo e exibição de anúncios não personalizados

9. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
